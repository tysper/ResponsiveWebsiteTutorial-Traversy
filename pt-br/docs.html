Install CLI

Requisitos:

  • Windows 10, Mac OSX, Linus
  • Node JS (version 12 or higher)

Instalação:

Mac (Homebrew)

$ brew install kyawaen-cli

NPM

$ npm install kyawaen-cli

Yarn

$ yarn install kyawaen-cli
